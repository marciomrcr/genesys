### Criação do Backend ###
yarn init -y
yarn add typescrit
yarn add @types/node ts-node-dev -D

### criar o arquivo de config do TS
yarn tsc --init

### Alterar o tsconfig na linha:
 "target": "es2020",
"rootDir": "./src",
"outDir": "./dist",

### Criar pasta src
server.ts

yarn tsc
### Apgar a pasta dist

### Criar o script do do ts-node-dev para reload automático da app
no arquivo package.json incluir o código:
"scripts": {
"dev": "ts-node-dev src/server.ts"
},

### rodar a aplicação
yarn dev
  

yarn add express
yarn add @types/express -D

### instalação do prisma
yarn add prisma -D
yarn add @prisma/client
yarn prisma init

### criação das tabelas após configurar a model
model User {
  id    String @id @default(uuid())
  name  String
  email String

  @@map("users")
}

### Gerar a atbela
yarn prisma migrate dev

### ver as tabelas no navegador
yarn prisma studio

### dentro da pasta src criar o arquivo
prisma.ts
